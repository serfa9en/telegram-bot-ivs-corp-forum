# Телеграм-бот для Цифрового форума 2024 Пермь

Бот имеет разный функционал для разного типа пользователей:
* админ (видит и контролит все, собирает статистику)
* спикер (видит список вопросов)
* участник форума (может выбрать спикера и задать любой вопрос, на который спикер даст развернутый ответ на соотвествующем мероприятии форума)
